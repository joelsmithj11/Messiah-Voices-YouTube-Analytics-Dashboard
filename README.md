# Messiah-Voices-YouTube-Analytics-Dashboard
## Project Overview

This project is a Power BI dashboard developed to analyze the performance of the Messiah Voices YouTube channel. The dashboard provides insights into video views, watch time, subscriber growth, impressions, click-through rate (CTR), and content performance trends. It helps identify top-performing videos and understand audience engagement over time.

## Features
- Total Views Analysis
- Watch Time Analysis
- Subscriber Growth Tracking
- Impressions and CTR Monitoring
- Top Performing Videos
- Channel Growth Trends
- Interactive Filters by Video Title and Upload Year
- Dynamic KPI Cards
- Tools and Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset
- Data Visualization Techniques

## Dataset Information
The dataset was generated from YouTube Analytics data of the Messiah Voices YouTube channel and contains:

- Video Title
- Upload Date
- Views
- Watch Time (Hours)
- Subscribers Gained
- Impressions
- Click Through Rate (CTR)

## Dashboard Insights
1. Identifies top-performing videos based on views and watch time.
2. Tracks channel growth across different years.
3. Monitors audience engagement using impressions and CTR.
4. Provides an overview of subscriber growth and content reach.

## Requirements

To open and run this project:

- Software Requirements
- Microsoft Power BI Desktop (Latest Version)
- Windows 10/11
- Dataset Requirements
- CSV file containing YouTube Analytics data

## How to Use
- Download the repository.
- Open the .pbix file using Power BI Desktop.
- Refresh the dataset if required.
- Interact with filters and visualizations to explore insights.
